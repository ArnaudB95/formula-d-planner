REGLES DU BLOC INFO - DASHBOARD
=======================================================================

DESCRIPTION
-----------
Le bloc info est la zone en haut a gauche du dashboard qui affiche
des messages rotatifs (source + texte).

Il existe 2 categories de messages :
1) Messages systeme
2) Messages fun

MESSAGES SYSTEME
----------------
Ordre des messages systeme :

1. Nombre de parties a venir
  Condition speciale: afficher ce message seulement si l'utilisateur
  connecte est "present" sur au moins une partie a venir.
  Exemple: "Parties 2 dates a venir" ou "Partie 1 date a venir"

2. Propositions en attente pour l'utilisateur
  Exemple: "Propositions 2 en attente" ou "Proposition 1 en attente"

3. Mentions @ de l'utilisateur dans le chat
  Exemple: "Chat 3 mentions @" ou "Chat 1 mention @"

4. Messages non lus du chat
  Exemple: "Chat 5 non lus" ou "Chat 1 non lu"

5. Autres membres en ligne
  Condition: afficher seulement si d'autres utilisateurs que moi sont en ligne.
  Exemple: "1 autre Pilote est en ligne"
  Exemple: "3 autres Pilotes sont en ligne"

Notes:
- Un message systeme n'apparait que si sa condition est vraie.
- Le pluriel s'adapte automatiquement.
- Structure: { source, text }

MESSAGES FUN
------------
Un message fun a la forme:
"[Pseudo] ..."

Regles des messages fun:
- Cible seulement les membres ayant un pseudo perso (pas d'email brut).
- Le pseudo est choisi aleatoirement.
- Un pseudo ne doit pas etre reutilise avant que tous les pseudos eligibles
  aient ete utilises une fois (cycle complet).
- Les templates fun sont aussi tires aleatoirement sans repetition,
  puis re-melanges une fois tous utilises.

CYCLE GLOBAL D'AFFICHAGE
------------------------
Cycle voulu:
- Toujours des messages systeme en premier
- Puis 1 message fun
- Puis de nouveau 1 a 2 messages systeme
- Puis 1 message fun
- Etc.

Details:
- Les messages systeme ont leur propre cycle sequentiel.
- Les messages fun ont leur propre cycle aleatoire sans repetition.
- Exemple avec 2 messages systeme:
  systeme 1 -> systeme 2 -> fun -> systeme 1 -> systeme 2 -> fun -> ...

TIMING ET ANIMATION
-------------------
- Intervalle de rotation: 4200 ms
- Duree du fondu: 240 ms
- Si un seul message total est disponible: affichage statique

RENDU
-----
- Bloc dans la barre du haut
- Texte compact et lisible
- Source coloree a gauche du texte
